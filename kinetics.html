8. chempy.kinetics — Kinetics Models¶

This module contains the kinetics models that are available in ChemPy. All such models derive from the KineticsModel base class.

8.1. Kinetics Models¶

class chempy.kinetics.KineticsModel(Tmin=0.0, Tmax=10000000000.0, Pmin=0.0, Pmax=1e+100, numReactants=-1, comment='')¶

Represent a set of kinetic data. The details of the form of the kinetic data are left to a derived class. The attributes are:

Attribute

Type

Description

Tmin

float

The minimum absolute temperature in K at which the model is valid

Tmax

float

The maximum absolute temperature in K at which the model is valid

Pmin

float

The minimum absolute pressure in Pa at which the model is valid

Pmax

float

The maximum absolute pressure in Pa at which the model is valid

numReactants

int

The number of reactants (used to determine the units of the kinetics)

comment

str

A string containing information about the model (e.g. its source)
getRateCoefficients(Tlist)¶

Return the rate coefficient k(T) in SI units at temperatures Tlist in K.

isPressureValid(P)¶

Return True if pressure P in Pa is within the valid pressure range, and False if not.

isTemperatureValid(T)¶

Return True if temperature T in K is within the valid temperature range and False if not.

class chempy.kinetics.ArrheniusModel(A=0.0, n=0.0, Ea=0.0, T0=298.15)¶

Represent a set of modified Arrhenius kinetics. The kinetic expression has the form

\[k(T) = A \left( \frac{T}{T_0} \right)^n \exp \left( - \frac{E_\mathrm{a}}{RT} \right)\]

where \(A\), \(n\), \(E_\mathrm{a}\), and \(T_0\) are the parameters to be set, \(T\) is absolute temperature, and \(R\) is the gas law constant. The attributes are:

Attribute

Type

Description

A

float

The preexponential factor in s^-1, m^3/mol*s, etc.

T0

float

The reference temperature in K

n

float

The temperature exponent

Ea

float

The activation energy in J/mol
changeT0(T0)¶

Changes the reference temperature used in the exponent to T0, and adjusts the preexponential accordingly.

fitToData(Tlist, klist, T0=298.15)¶

Fit the Arrhenius parameters to a set of rate coefficient data klist corresponding to a set of temperatures Tlist in K. A linear least- squares fit is used, which guarantees that the resulting parameters provide the best possible approximation to the data.

getRateCoefficient(T, P=100000.0)¶

Return the rate coefficient k(T) in SI units at temperature T in K.

class chempy.kinetics.ArrheniusEPModel(A=0.0, E0=0.0, n=0.0, alpha=0.0)¶

Represent a set of modified Arrhenius kinetics with Evans-Polanyi data. The kinetic expression has the form

\[k(T) = A T^n \exp \left( - \frac{E_0 + \alpha \Delta H_\mathrm{rxn}}{RT} \right)\]

The attributes are:

Attribute

Type

Description

A

float

The preexponential factor in s^-1, m^3/mol*s, etc.

n

float

The temperature exponent

E0

float

The activation energy at zero enthalpy of reaction in J/mol

alpha

float

The linear dependence of activation energy on enthalpy of reaction
getActivationEnergy(dHrxn)¶

Return the activation energy in J/mol using the enthalpy of reaction dHrxn in J/mol.

getRateCoefficient(T, dHrxn)¶

Return the rate coefficient k(T, P) in SI units at a temperature T in K for a reaction having an enthalpy of reaction dHrxn in J/mol.

toArrhenius(dHrxn)¶

Return an ArrheniusModel object corresponding to this object by using the provided enthalpy of reaction dHrxn in J/mol to calculate the activation energy.

class chempy.kinetics.PDepArrheniusModel(pressures=None, arrhenius=None)¶

A kinetic model of a phenomenological rate coefficient k(T, P) using the expression

\[k(T,P) = A(P) T^{n(P)} \exp \left[ \frac{-E_\mathrm{a}(P)}{RT} \right]\]

where the modified Arrhenius parameters are stored at a variety of pressures and interpolated between on a logarithmic scale. The attributes are:

Attribute

Type

Description

pressures

list

The list of pressures in Pa

arrhenius

list

The list of ArrheniusModel objects at each pressure
fitToData(Tlist, Plist, K, T0=298.0)¶

Fit the pressure-dependent Arrhenius model to a matrix of rate coefficient data K corresponding to a set of temperatures Tlist in K and pressures Plist in Pa. An Arrhenius model is fit at each pressure.

getRateCoefficient(T, P)¶

Return the rate constant k(T, P) in SI units at a temperature Tlist in K and pressure P in Pa by evaluating the pressure- dependent Arrhenius expression.

class chempy.kinetics.ChebyshevModel(Tmin=0.0, Tmax=0.0, Pmin=0.0, Pmax=0.0, coeffs=None)¶

A kinetic model of a phenomenological rate coefficient k(T, P) using the expression

\[\log k(T,P) = \sum_{t=1}^{N_T} \sum_{p=1}^{N_P} \alpha_{tp} \phi_t(\tilde{T}) \phi_p(\tilde{P})\]

where \(\alpha_{tp}\) is a constant, \(\phi_n(x)\) is the Chebyshev polynomial of degree \(n\) evaluated at \(x\), and

\[\tilde{T} \equiv \frac{2T^{-1} - T_\mathrm{min}^{-1} - T_\mathrm{max}^{-1}} {T_\mathrm{max}^{-1} - T_\mathrm{min}^{-1}}\]
\[\tilde{P} \equiv \frac{2 \log P - \log P_\mathrm{min} - \log P_\mathrm{max}} {\log P_\mathrm{max} - \log P_\mathrm{min}}\]

are reduced temperature and reduced pressures designed to map the ranges \((T_\mathrm{min}, T_\mathrm{max})\) and \((P_\mathrm{min}, P_\mathrm{max})\) to \((-1, 1)\). The attributes are:

Attribute

Type

Description

coeffs

list

Matrix of Chebyshev coefficients

degreeT

int

The number of terms in the inverse temperature direction

degreeP

int

The number of terms in the log pressure direction
fitToData(Tlist, Plist, K, degreeT, degreeP, Tmin, Tmax, Pmin, Pmax)¶

Fit a Chebyshev kinetic model to a set of rate coefficients K, which is a matrix corresponding to the temperatures Tlist in K and pressures Plist in Pa. degreeT and degreeP are the degree of the polynomials in temperature and pressure, while Tmin, Tmax, Pmin, and Pmax set the edges of the valid temperature and pressure ranges in K and Pa, respectively.

getRateCoefficient(T, P)¶

Return the rate constant k(T, P) in SI units at a temperature Tlist in K and pressure P in Pa by evaluating the Chebyshev expression.
